# Binary Boutique
Website URL: https://binaryboutique.netlify.app
A modern fashion e-commerce platform curating timeless fashion for the modern individual.

## About
Binary Boutique offers a carefully curated selection of fashion pieces that celebrate unique style and confidence. Our platform features:

- Product catalog with detailed views
- Event showcase and ticketing
- Board of Trustees information
- Appointment booking system
- Contact and customer support

## Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)
- Responsive Design
